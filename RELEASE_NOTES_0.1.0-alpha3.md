# Bard 0.1.0-alpha3

Bard is a minimalist, text-only audiobook player designed for the Light Phone
III. This private alpha brings local, Libby, and RSS audiobooks into one calm
Books screen and shared player.

- Connect an existing Libby account using Libby's eight-digit Copy To Another
  Device setup code, then listen through Bard's hidden persistent player.
- Scan single-file MP3 and M4B audiobooks placed directly in the device's
  `Audiobooks` folder.
- Add RSS feeds with MP3 or M4B enclosures, stream books, and explicitly keep RSS
  books offline in Bard.
- Resume saved listening position and speed, with recent books ordered by last
  played time.
- Continue Libby, local, and RSS playback while navigating Bard or while the
  screen is off. Libby playback displays an Android foreground notification.

This is alpha software. Expect rough edges and keep copies of important local
files. Local multi-file books and chapters are not supported. RSS feeds that
require authentication are not supported. Libby integration depends on Libby's
web application and may need maintenance as it changes.

Bard is an independent project and is not affiliated with, endorsed by, or
sponsored by Libby or OverDrive.
